# 国际化

国际化使用详情，参考plugin-locale (opens new window)。

唯一要注意的是页面的 title。我们依旧以登录页面（/src/pages/login/index.tsx）为例：

function Login() {
  ...
  ...
}

// title 在应用运行时，会被 src/locales/ 下对应语言文件里的内容替换
Login.title = 'LOGIN_TITLE'

export default Login
